What this app does

This workspace centralizes day-to-day operations for faith-integrated counseling practices. It includes role-aware views, clinical workflows, scheduling, client portal interactions, and billing/reporting workflows backed by API endpoints.

Core Modules

  • Client Lifecycle & Charting

    Client records, treatment plans, progress notes, consents, intake packets.

  • Scheduling & Operations

    Appointments, reminders, waitlist, staffing, and availability support.

  • Billing & Claims

    Service codes, fee schedules, invoices, payments, superbills, claims, aging.

  • Portal & Faith Workflows

    Portal messaging/documents plus faith resources, goals, and inventory workflows.

Platform & Docs

  • API Health

    Health checks and telemetry summaries for API and web services.
