# 10 — Offerings and Financials

**Faith Counseling User Manual**

---

## Overview

The Offerings module tracks the service arrangements and financial agreements the practice maintains with its clients. It provides a structured record of how each client is being served and under what financial terms.

---

## 10.1 Accessing Offerings

Offerings are managed from **Workspace Studio** → **Offerings** tab. This surface is available to practice owners, practice admins, and platform admins.

---

## 10.2 What an Offering Record Tracks

Each client can have one or more offering records. An offering record captures:

| Field | Description |
|---|---|
| **Client** | The client associated with this offering |
| **Service Type** | The counseling service being offered (individual, group, family, ministry, telehealth) |
| **Frequency** | How often the service is provided (weekly, bi-weekly, monthly, as needed) |
| **Financial Arrangement** | How this service is being paid for (see below) |
| **Start Date** | When this arrangement began |
| **End Date** | When this arrangement ended (leave blank if ongoing) |
| **Notes** | Any additional context about this offering |

---

## 10.3 Financial Arrangement Types

| Type | Description |
|---|---|
| **Standard Fee** | Full market rate session fee |
| **Sliding Scale** | Fee adjusted based on client income or ability to pay |
| **Pro Bono** | No charge — service provided as a ministry or community offering |
| **Insurance Billed** | Service billed through the client's insurance carrier |
| **Scholarship / Grant** | Fee covered through a practice scholarship or grant program |

---

## 10.4 Creating an Offering Record

1. Open **Workspace Studio** → **Offerings**.
2. Click **Add Offering**.
3. Select the **client** from the dropdown.
4. Choose the **service type** and **frequency**.
5. Select the **financial arrangement**.
6. Set the **start date**.
7. Add any relevant notes.
8. Click **Save**.

---

## 10.5 Editing an Offering

1. Click the offering record from the list.
2. Update the relevant fields.
3. If the arrangement has ended, set an **end date**.
4. Click **Save**.

---

## 10.6 Offering History

All offering records — including ended arrangements — remain in the history for each client. This provides a complete picture of how the client has been served financially over time.

---

## 10.7 Faith-Based Financial Practices

Faith Counseling recognizes that many Christian counseling practices operate under a ministry model. The offering module fully supports:

- **Pro bono** and **scholarship** tracks for ministry-funded clients
- **Sliding scale** arrangements reflective of compassion-based fee structures
- Notes fields for capturing the pastoral or ministerial context of specific arrangements
