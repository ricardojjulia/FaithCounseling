# 02 — Dashboard and Home

**Faith Counseling User Manual**

---

## Overview

The Dashboard is the daily command center for counselors and practice administrators. It surfaces the most important signals across your current caseload, scheduling posture, note compliance, and portal activity in one place.

---

## 2.1 Counselor Home

When a counselor logs in, the **Counselor Home** screen provides a personalized summary:

### Summary Cards

| Card | What It Shows |
|---|---|
| **Active Clients** | Total active client count for this counselor |
| **Today's Appointments** | Scheduled sessions for the current day |
| **Note Gaps** | Clients with notes overdue by 1 day, 3 days, or 7+ days |
| **Outstanding Assignments** | Total form and document assignments awaiting client action |
| **Faithful Workflows** | Count of flagged or high-priority workflow recommendations |

Clicking a **Faithful Workflows** metric card navigates directly into the [Faithful Workflows](06-faithful-workflows.md) workspace, matching the drill-down behavior of other metric cards.

### Quick Actions

From the Counselor Home, use the quick action buttons to:

- **Open Scheduling** — jump to the scheduling calendar
- **View Clients** — open the full client list
- **Open a Client Record** — select and open a specific client
- **Open Clinical Chart** — go directly to a client's chart
- **Open Documents** — navigate to client document assignments

---

## 2.2 Operations Dashboard (Admin View)

Practice administrators and owners see the **Operations Dashboard**, a live operational summary covering the full practice.

### What the Dashboard Shows

| Section | Details |
|---|---|
| **Counselor Workload** | Per-counselor client count and load distribution |
| **Note-Gap Compliance Watch** | Clients with session notes missing beyond 1-day, 3-day, and 7-day thresholds |
| **Portal Request Tracking** | Pending portal registration and care requests requiring staff action |
| **7-Day Trend Context** | Historical trend lines for key operational metrics |
| **Configurable Operational Alerts** | Flags for overdue tasks, capacity thresholds, or compliance states |

---

## 2.3 Note-Gap Compliance

The note-gap compliance feature helps practices stay current with session documentation requirements.

Three tier levels are tracked:

| Tier | Meaning |
|---|---|
| **Over 1 Day** | Client session note is 1+ days overdue |
| **Over 3 Days** | Client session note is 3+ days overdue |
| **Over 7 Days** | Client session note is 7+ days overdue — most urgent |

Note-gap counts appear on the Dashboard and the Faithful Workflows banner so both surfaces stay aligned.

---

## 2.4 Outstanding Assignments

The Outstanding Assignments summary shows:

- **Total** assignments across all clients
- **Documents** — forms and documents waiting for client submission
- **Forms** — intake packets or assigned forms pending completion

---

## 2.5 Portal Request Tracking

New portal activity creates actionable items visible on the dashboard:

- **Account signup requests** — new clients requesting access to the portal
- **Care requests** — client-submitted requests for counseling services

Practice admins can navigate directly from the dashboard to the **Workspace Studio Portal** tab to approve, convert, or manage these requests.

---

## 2.6 Dashboard Navigation Shortcuts

Each metric card on the dashboard is a navigation shortcut:

| Card | Navigates To |
|---|---|
| Clients card | Client list filtered to active clients |
| Appointments card | Scheduling calendar for today |
| Faithful Workflows card | Faithful Workflows workspace |
| Note Gaps card | Client list filtered for note-gap clients |
| Portal Requests card | Workspace Studio → Portal tab |
