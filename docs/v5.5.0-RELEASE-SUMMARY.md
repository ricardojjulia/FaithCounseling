## What's New

### Faithful Workflows — Counselor-Facing Care Recommendation Engine

A three-panel workspace accessible from the sidebar "Faith Workflows" link that reviews each client's clinical data and surfaces prioritized, explainable care recommendations.

Always-visible safety banner above all panels. Critical-state variant (red filled) when any client has critical urgency.

## Recommendation Engine

27 deterministic rules across 7 categories. Rules fire against structured client data — no black-box AI. Rationale is template-interpolated with actual data values (e.g., "PHQ-9 = 18, above threshold of 15 — worsening trend 12→16→18").

| Category | Rules |
|----------|-------|
| Safety | PHQ-9 severe (≥20), suicidal ideation (item 9 ≥2), PCL-5 crisis (≥51), consecutive no-shows, risk keywords in notes |
| Clinical Cautions | PHQ-9 worsening trend, GAD-7 high (≥15), no treatment plan, stale plan (>90d), no recent note (>30d), dx without goal |
| Session Focus | Overdue treatment goals, pending homework, pending assessment |
| Homework | No between-session homework in 2 sessions, journaling suggestion |
| Relationship | Systems goal missing for couples/family therapy, relationship concern in note |
| Spiritual (all optional) | Biblical integration, faith-based grief support, transition prayer, faith acknowledge |
| Coordination | No insurance on file, open referral (>21d), faith referral, closing summary needed |
| Monitoring | Reassessment overdue (>90d), discharge planning (all goals met), stable progress |

### Urgency Scoring (0–100)

| Score | Level | Color |
|-------|-------|-------|
| 70–100 | Critical | Red |
| 40–69 | High | Orange |
| 20–39 | Moderate | Yellow |
| 0–19 | Routine | Gray |

## Safety Invariants

1. Safety nodes with priority ≥ 9 **cannot be hidden or deferred**
2. Safety nodes always rendered first regardless of sort
3. Spiritual nodes always labeled "(Optional)" — never outrank safety/clinical
4. Safety banner is visible in all page states
5. All action outputs watermarked: "AI-assisted draft — final judgment belongs to the counselor"
6. No spiritual content generated for clients without a faith profile
7. No language implies symptoms are caused by weak faith
