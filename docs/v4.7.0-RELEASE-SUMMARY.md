# v4.7.0 Release Summary

## Expanded Counseling Form Library + Operations Dashboard

**Release Date:** March 29, 2026  
**Version:** 4.7.0  
**Status:** Production Ready  
**Scope:** Documents library expansion and operations dashboard upgrade

## What Is Included

### Form Library Expansion

- Expanded the Documents module from 19 forms to 39 forms
- Added 20 new forms covering:
  - consent and administrative workflows
  - deeper clinical assessments
  - treatment-planning templates
  - therapeutic worksheets
  - faith-integrated reflection tools

### Operations Dashboard Upgrade

- Replaced placeholder dashboard cards with a real operations summary
- Added counselor workload bars and 1-hour gap visibility for Today's Schedule
- Added a real Priority Queue based on the `highTouchpoint` client flag
- Added Compliance Watch metrics for note gaps and incomplete assigned work
- Refocused the Clients card on totals, unscheduled clients, and portal request statuses
- Added dashboard drill-down workflows so staff can open client, scheduling, documents, and portal-review queues directly from the dashboard
- Added operational alert thresholds for unscheduled high-touchpoint clients, note-gap backlog, exhausted same-day counselor capacity, and portal backlog growth
- Added 7-day dashboard trends for utilization, documentation backlog, portal flow, and unscheduled-client backlog
